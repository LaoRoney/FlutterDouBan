# dev-open-push
   
   ### 大家可以向这个分支根据豆瓣UI做设计稿，提交代码

   * 这个分支供广大Flutter开发者来学习Flutter
   * **此分支囊括了Flutter 90%的组件的基本与组合使用**，是初学者真正实践的不错选择
   * 在此豆芽APP的首页实现与影片详情的UI特效，基于魔改Flutter源码。有兴趣可以看看
   * 大家想实战自己的Flutter能力，可以将某瓣APP作为设计稿，完成需求，并提交
   * 每位开发者提交的代码，我都会在文档中进行备注
